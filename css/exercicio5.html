CSS: Exercício 5

Este exercício serve de introdução para os pseudo-elementos em CSS.

Conforme a descrição da MDN,

"Um pseudo-elemento CSS é uma palavra-chave adicionada a um seletor que permite que você estilize uma parte específica do elemento selecionado."

É interessante notar que no parágrafo anterior, as aspas foram incluídas na formatação do pseudo-elemento, ou seja, ele realmente pega a primeira letra, não apenas o primeiro caractere.
